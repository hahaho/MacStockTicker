# MacStockTicker 📈

一个专为 Mac 摸鱼/高效盯盘设计的极简、透明、悬浮的 A 股/指数实时监控小工具。

基于 Swift 和 SwiftUI 原生开发，内存占用极低，支持经典卡片面板与“跑马灯”新闻条两种模式无缝切换。

## ✨ 核心特性

- **极致透明与悬浮**：完美融合 Mac 桌面，支持深色模式下的毛玻璃（Glassmorphism）质感，悬浮在所有窗口最上层，不打扰正常工作。
- **双模自由切换**：
  - **展开模式**：显示详细的高/低/开/量等指标，支持列表滚动，信息一目了然。
  - **跑马灯模式（摸鱼模式）**：一键最小化为仅 40px 高的顶部/底部长条，60fps 丝滑无缝滚动展示。
- **实时数据**：接入新浪财经 API，毫秒级响应，无需 API Key，默认 5 秒自动刷新。
- **智能配置与记忆**：
  - **界面直装**：支持直接在面板中输入代码添加股票，鼠标悬浮在股票上可快速删除。
  - **自动识别**：自动识别沪（sh）深（sz）前缀。
  - **持久化**：自动持久化记忆最后一次查询的股票列表及窗口（展开/最小化）状态。
- **大盘常驻**：默认置顶显示 上证指数、深证成指、创业板指。

## 🚀 快速开始

### 1. 环境要求
- macOS 11.0 或更高版本
- 随系统自带的 `swiftc` 编译器 (通常安装了 Xcode Command Line Tools 即可)

### 2. 打包与安装 (推荐)

克隆或下载本仓库到本地：

```bash
git clone https://github.com/hahaho/MacStockTicker.git
cd MacStockTicker
```

运行一键打包安装脚本：

```bash
chmod +x build_app.sh
./build_app.sh
```

脚本会自动将代码编译为标准的 macOS `MacStockTicker.app` 应用程序。在脚本执行最后，您可以选择输入 `y` 将其自动安装到 `/Applications`（应用程序）目录。之后您就可以通过 Spotlight (Cmd + Space) 搜索 `MacStockTicker` 像普通软件一样打开它了！

*(注意：首次打开可能会有安全提示，只需在系统设置 -> 隐私与安全性中允许即可)*

### 3. 脚本直接运行 (开发者模式)

如果您不想安装 App，也可以直接通过脚本运行：

```bash
chmod +x run.sh
./run.sh
```

## 🛠 操作手册

### 界面交互
- **添加股票**：在展开模式下，直接在输入框中输入 6 位股票代码（如 `600519`），点击 `+` 号或按回车键即可添加。
- **删除股票**：将鼠标悬停在对应的股票卡片上，右侧会出现红色的删除按钮 🗑，点击即可移除。
- **退出程序**：点击界面右上角（或跑马灯最右侧）的 🔴 `关闭` 按钮。
- **切换跑马灯/展开模式**：点击 ➖ `最小化` 按钮进入跑马灯；点击 ➕ `展开` 按钮恢复面板。
- **拖动位置**：点击窗口的透明空白处即可任意拖拽（即使在全屏其他 App 也能保持置顶）。

### 代码架构简介
- `StockTicker.swift`：核心逻辑，包含 SwiftUI 界面、WindowManager（窗口状态管理）、MarqueeView（跑马灯引擎）和新浪 API 解析。
- `run.sh`：自动化编译与启动脚本。
